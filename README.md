# Polars ETL Demo - YZV 322E

## 1. What is this tool?
Polars is a lightning-fast, Rust-based DataFrame library for Python. It provides a highly efficient, multi-threaded, and lazy-evaluation alternative to Pandas, making it ideal for processing large datasets in modern ETL pipelines.

## 2. Prerequisites
- Operating System: Windows (WSL2 recommended), macOS, or Linux
- Docker and Docker Compose installed (Docker version 20.10+ recommended)
- Git (to clone the repository)

## 3. Installation
Open your terminal and clone this repository using the following commands:
```bash
git clone [https://github.com/KULLANICI_ADIN/polars-etl-demo.git](https://github.com/KULLANICI_ADIN/polars-etl-demo.git)
cd polars-etl-demo
